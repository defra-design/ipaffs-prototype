Commodity code

Required for all commodities.

Enter a commodity code. You can look up commodity codes (opens in new tab) .

EPPO code

Required for all commodities (unless you enter genus and species instead).

You can search for EPPO codes on the EPPO global database (opens in new tab) .

Genus and species

Required for all commodities (unless you enter an EPPO code instead).

A valid scientific name for the commodity, such as “Chamaecrista comosa var. capricorna”

It must match names in reference data for IPAFFS.

You can enter an EPPO code instead and leave genus and species blank

Variety

Required for Fruit and Vegetables (commodity codes starting with 07 and 08 ) which are HMI regulated. {#

There are 286 varieties. You can check varieties when you look up commodity code (opens in new tab).

#}

Class

Required for Fruit and Vegetables (commodity codes starting with 07 and 08) which are HMI regulated.

Your answer must match one of these values exactly (including spaces and capital letters):

  • Class II
  • Class I
  • Extra Class

Intended use

Required for commodities that can be regulated by both the Plant Health and Seeds Inspectorate (PHSI) and the Horticultural Marketing Inspectorate (HMI).

Your answer must match one of these values exactly (including spaces and capital letters):

  • For planting or planted in a growing substrate only
  • For sowing
  • Planted in a growing substrate only
  • Seeds for sowing

Number of packages

Required for all commodities.

Enter the number of packages for a commodity in your consignment. You must use digits.

Warning Do not use commas to separate parts of this number. For example: enter one thousand packages as 1000.

Required for all commodities.

Net weight

Required for all commodities.

The net weight of this entire commodity line, measured in kilograms (KG).

!

Warning Do not use commas to separate parts of this number. For example: enter one thousand kilograms as 1000.
