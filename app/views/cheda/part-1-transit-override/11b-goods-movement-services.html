Is your consignment using the Common Transit Convention (CTC)?

It is 18 characters, starting with 2 numbers, a 2-letter country code then 14 letters and numbers. For example, 24GB123456789AB012.
What is the CTC?

The Common Transit Convention (CTC) is a customs procedure that can help you move goods quickly between:

  • the UK
  • the EU
  • other CTC countries

To move your goods, you need to:

  1. Check if you can use transit (opens in new tab)
  2. Find out how to move goods using transit (opens in new tab)
  3. Plan your route before using transit (opens in new tab)

Using transit can help reduce the number of customs declarations you’ll need to complete.

Will the transport use the Goods Vehicle Movement Service (GVMS)?

What is the GVMS?

The GVMS helps to speed up customs clearance at some UK ports.

To move your goods, you need to:

  1. Find out which ports use the GVMS (opens in new tab)
  2. Register for the GVMS (opens in new tab)
  3. Get a goods movement reference (GMR) (opens in new tab)

The GMR is a single reference that brings together multiple declarations. Without it, you cannot move your goods through UK ports which use the GVMS and you may face delays.

The driver must check the GVMS to find out if an inspection is required before they disembark.
